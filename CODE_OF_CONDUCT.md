# Code of Conduct

Console adopts the **Contributor Covenant**, version 2.1, as its code of conduct.

The full text is maintained by the Contributor Covenant project and is available
here:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: this project is a welcoming, harassment-free space for everyone,
regardless of background or identity. Be respectful, assume good faith, and help
keep the community healthy. Maintainers may remove, edit, or reject contributions
and comments that are not aligned with this code, and may temporarily or
permanently ban any contributor for behavior they deem inappropriate.

## Scope

This code applies in all project spaces (the repository, issues, pull requests,
and discussions) and when an individual is representing the project in public
spaces.

## Reporting

To report unacceptable behavior, contact the maintainers at:

- **conduct@refringence.ai**

All reports will be reviewed and investigated promptly and fairly. Maintainers are
obligated to respect the privacy and security of the reporter of any incident.
